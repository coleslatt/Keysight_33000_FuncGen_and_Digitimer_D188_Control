Kt33000OutputChannelOutputList¶

Access interface members using: driver.output_channels[<name_or_index>].output.list
Where driver is a reference to the driver’s root interface returned by the constructor.
class Kt33000OutputChannelOutputList¶

Kt33000OutputChannelOutputList interface.

property dwell_time¶

Sets dwell time, the amount of time each frequency in a frequency list is generated.

Access:

ReadWrite

Type:

datetime.timedelta

Remarks:
•Parameter : 1 µs to 1000 s, default 1 s.
•Typical Return : +1.200000000000000E+01
•The instrument generates each frequency in a frequency list for the specified dwell time, when TriggerSource is Immediate.
property frequency_points¶

Read Only. Returns number of frequencies in current frequency list.

Access:

ReadOnly

Type:

int

Remarks:
•Parameter : MAXimum or MINimim.
•Typical Return : +17
•The default list has three frequencies: 100 Hz, 1000 Hz, and 550 Hz.
property frequency_points_max¶

Read Only. The maximum number of frequencies that are in the current frequency list for the specified channel.

Access:

ReadOnly

Type:

int

Remarks:
MAXimum is 128.
property frequency_points_min¶

Read Only. The minimum number of frequencies that are in the current frequency list for the specified channel.

Access:

ReadOnly

Type:

int

Remarks:
MINimum is 1.
