BurstGatePolarity¶

class BurstGatePolarity¶

Selects burst gate polarity either normal or inverted.

Members:

NORMAL : Select true-high logic for an externally gated burst. This is default Burst Gate polarity.

INVERTED : Select true-low logic for an externally gated burst.
