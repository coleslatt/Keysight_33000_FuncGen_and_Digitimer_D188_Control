Kt33000OutputChannelOutputVoltageLimit¶

Access interface members using: driver.output_channels[<name_or_index>].output.voltage.limit
Where driver is a reference to the driver’s root interface returned by the constructor.
class Kt33000OutputChannelOutputVoltageLimit¶

Kt33000OutputChannelOutputVoltageLimit interface.

property enabled¶

Enables or disables output amplitude voltage limits.

Access:

ReadWrite

Type:

bool

Remarks:
•Parameter : 0 (OFF) or 1 (ON). Default 0 (OFF).
•When this is turned ON, if the present settings of amplitude and offset exceed the limits, then the limits will be disabled. The instrument will generate either a "Settings conflict" error.
•When VoltageCouplingEnabled is ON, and VoltageLimitEnabled is also set to ON, voltage limit settings on both channels affect maximum amplitude and offset voltage settings on both channels. The most restrictive combination of high and low limits from either channel is used.
•Limits are set in reference to the current setting of OutputLoad. If the specified LOAD impedance is not present at the instrument's output, then the output limit may not represent the actual voltages at the output connector. For example, if the output impedance is set to 50 Ω, but the actual load is high impedance, then the actual output peak voltage may be up to twice the specified limit voltage.

property high¶

The positive limit voltage in volts for the output amplitude of the specified channel.

Access:

ReadWrite

Type:

float

Remarks:
•Parameter : ±5 VDC into 50 Ω, as long as HIGH is at least 1 mV greater than LOW.
•Defaults: +50 mV.
•Typical Return: +5.0000000000000E+00.
•For voltage limits to be in effect, VoltageLimitEnabled must be ON. If this is the case, and the high limit is set below the high value of the signal or the low limit is set above the low value of the signal, the relevant limit will be clipped to the high or low value of the signal. The instrument will generate either a "Data out of range" or "Settings conflict" error.
•The high limit sets the highest output voltage allowed to be set, including DC Offset and peak amplitude. It is set in reference to the current OutputLoad setting. If the specified LOAD impedance is not present at the instrument's output, then the output limit may not represent the actual voltages at the output connector. For example, if the output impedance is set to 50 Ω, but the actual load is high impedance, then the actual output peak voltage may be up to twice the specified limit voltage.
•Specifying Voltage Units: You can set the output limit voltage only in volts.
•When VoltageLimitEnabled is ON, voltage limit settings on both channels affect maximum amplitude and offset voltage settings on both channels. The most restrictive combination of high and low limits from either channel is used.
