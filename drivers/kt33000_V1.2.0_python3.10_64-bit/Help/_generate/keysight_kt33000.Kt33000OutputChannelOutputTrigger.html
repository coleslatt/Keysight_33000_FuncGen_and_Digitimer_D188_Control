Kt33000OutputChannelOutputTrigger¶

Access interface members using: driver.output_channels[<name_or_index>].output.trigger
Where driver is a reference to the driver’s root interface returned by the constructor.
class Kt33000OutputChannelOutputTrigger¶

Kt33000OutputChannelOutputTrigger interface.

property enabled¶

Disables (OFF) or enables (ON) the “trigger out” signal for sweep and burst modes.

Access:

ReadWrite

Type:

bool

Remarks:
•Parameter : 0 (OFF) or 1 (ON). Default 0 (OFF).
•When enabled, a pulse with the specified edge direction (OutputTriggerSlope) is output from the rear-panel Ext Trig connector at the beginning of the burst or sweep.
•In triggered burst mode:
 •With TriggerSource Immediate, instrument outputs square wave with 50% duty cycle from the Ext Trig connector. The waveform period equals the burst period (BurstInternalPeriod).
 •With TriggerSource External or BurstMode set to Gated, the instrument disables "trigger out." The rear-panel Ext Trig connector cannot be used for both operations simultaneously (an externally-triggered waveform uses the same connector to trigger sweep or burst).
 •With TriggerSource Bus, the instrument outputs a pulse (>1 μs pulse width on the EDU33210 Series and 33500 Series, and 100 ns on the 33600 Series) from the Ext Trig connector at the beginning of each sweep or burst.
•In frequency sweep mode:
 •With TriggerSource Immediate, the instrument outputs a square wave with a 50% duty cycle (the rising edge is the sweep trigger) from the Ext Trig connector. Waveform period equals to the sweep time (SweepTime).
 •With TriggerSource External, instrument disables the "trigger out" signal. The rear-panel Ext Trig connector cannot be used for both operations simultaneously (an externally-triggered sweep uses the same connector to trigger the sweep).
 •With TriggerSource Bus, the instrument outputs a pulse (>1 μs pulse width on the EDU33210 Series and33500 Series, and 100 ns on the 33600 Series) from the Ext Trig connector at the beginning of each sweep or burst.
property slope¶

Selects whether the instrument uses the rising edge or falling edge for the “trigger out” signal.

Access:

ReadWrite

Type:

TriggerSlope

Remarks:
•Parameter : Positive or Negative. Default Positive.
•Typical Return : Positive or Negative.
•Positive outputs a rising edge pulse; Negative outputs a falling edge pulse.
•When enabled using OutputTriggerEnabled , a pulse with the specified edge direction is output from the rear-panel Trig Out connector at the beginning of a sweep or burst.
property source¶

Selects the source channel used by trigger output on a two-channel instrument.

Access:

ReadWrite

Type:

TriggerSource

Remarks:
•Parameter : Channel1 or Channel2. Default Channel1.
•Typical Return : Channel1 or Channel2.
•In a two-channel instrument, either channel may be source channel for the trigger output, but channel 2 is not valid for dual arbitrary (IQ) waveforms.
