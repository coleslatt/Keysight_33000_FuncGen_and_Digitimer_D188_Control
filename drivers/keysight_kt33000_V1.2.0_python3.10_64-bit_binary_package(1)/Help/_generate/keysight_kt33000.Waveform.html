Waveform¶

class Waveform¶

Shape of Modulating waveform.

Members:

SINUSOID : Select a sinusoid waveform. This is default waveform shape.

SQUARE : Select a square waveform with a 50% duty cycle.

RAMP : Select a ramp waveform with 100% symmetry.

N_RAMP : Select a ramp waveform with 0% symmetry.

TRIANGLE : Select ramp waveform with 50% symmetry.

NOISE : Select Noise as a modulating wave shape.

PRBS : Select periodic random noise modulation. This option is not supported on 33210 series.

ARB : Select an arbitrary waveform as the modulating wave shape. This option is not supported on 33210 series.

USER : On setting User, an arbitrary waveform gets selected as the modulating waveshape. This option is supported only on 33210 series and for AM, FM and PWM .
