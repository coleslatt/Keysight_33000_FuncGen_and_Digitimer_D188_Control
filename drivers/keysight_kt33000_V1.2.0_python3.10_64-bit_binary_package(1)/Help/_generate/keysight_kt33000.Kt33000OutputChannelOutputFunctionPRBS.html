class Kt33000OutputChannelOutputFunctionPRBS¶

Kt33000OutputChannelOutputFunctionPRBS interface.

property bit_rate¶

Sets the pseudo-random binary sequence (PRBS) bit rate.

Access:

ReadWrite

Type:

int

Remarks:
•Parameter : 1 mbit/s to maximum allowed for the instrument, default 1000 bit/s.
•Typical Return : +1.920000000000000E+04
•A PRBS waveform using polynomial PNx is generated by a shift register of x bits, and the output waveform begins with x sample periods of high output. Sample period is the reciprocal of the sample rate, and the channel's Sync pulse indicates the waveform's start. For example, if the PRBS uses PN23 with sample rate 500 Hz, the output begins with 46 ms of high output (23 x 2 ms).
•The bit rate is independent of the data sequence length.
property data_format¶

Sets the pseudo-random binary sequence (PRBS) type. Setting the sequence type sets the length and feedback values.

Access:

ReadWrite

Type:

PRBSSequenceType

Remarks:
•Parameter : EDU33210 series and 33500 Series: PN7, PN9, PN11, PN15, PN20 or PN23
33600 Series: PN3 through PN32 (all 30 possible values from 3 to 32)
•Typical Return : PN# (PN3 through PN9) or PN## (PN10 through PN32), where each # is one digit.
•SYNC output may be active during a PRBS function, unlike the Noise function. The SYNC output marks the first data bit of a PRBS waveform.
•A PRBS waveform using polynomial PNx is generated by a shift register of x bits, and the output waveform begins with x sample periods of high output. Sample period is the reciprocal of the sample rate (PRBSBitRate), and the channel's Sync pulse indicates the waveform's start. For example, if the PRBS uses PN23 with sample rate 500 Hz, the output begins with 46 ms of high output (23 x 2 ms).
