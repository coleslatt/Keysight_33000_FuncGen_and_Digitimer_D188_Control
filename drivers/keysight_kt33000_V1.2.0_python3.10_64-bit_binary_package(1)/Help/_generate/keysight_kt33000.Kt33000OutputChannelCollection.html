Kt33000OutputChannelCollection¶

Access interface members using: driver.output_channels
Where driver is a reference to the driver’s root interface returned by the constructor.
class Kt33000OutputChannelCollection¶

Collection interface for the OutputChannelrepeated capability.

Note

Kt33000OutputChannelCollection is a Repeated Capability Collection class, refer to Repeated Capability Collections for instance access operations. Instance Names: OutputChannel1, OutputChannel2, Dynamic Instance Names

property count¶

The number of repeated capability instances.
